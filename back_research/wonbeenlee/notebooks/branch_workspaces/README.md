# Branch Workspaces

이 디렉토리는 `churn power` 모델의 3개 브랜치 작업 공간을 모아 둔다.

- `commerce_value_activity/`
  - 구매, 주문, 매출, 활동 감소 신호 중심
- `membership_subscription_lifecycle/`
  - 가입, tenure, renewal, expiry, lifecycle 신호 중심
- `trust_support_intervention/`
  - support, review, complaint, intervention 신호 중심

각 브랜치의 세부 기준은 `../model_branch_grouping.md` 와 `docs/prds/06~09` 문서를 따른다.
